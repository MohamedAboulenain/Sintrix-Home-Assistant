# Scripts

> Sintrix maintains this folder. Use for reusable scripts/scenes shared across automations or
> dashboards. Simple per-domain scripts can live in the domain file's **As-built** section.

## Spec template

```markdown
# Script — <name>

## Intent
What it does and where it's used (dashboards, automations).

## Sequence
Steps, fields/variables, native constructs.

## As-built
ha-mcp script/scene id(s) created/updated. Link callers.

## Open items
```
